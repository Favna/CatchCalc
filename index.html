Introduction

CatchCalc is a simple in Java written program that can calculate the chance of capture a pokémon given the pokémon Max HP and current HP. While a couple of steps are still required to actually know these values (explained below), it is wishful thinking to hope that in a future game we'll actually be able to see values rather than green bars. To utilize CatchCalc best I highly recommend you carry a Pokémon that knows the move False Swipe or Hold Back to put the opposing Pokémon at 1 HP!!

CatchCalc Instructions

In order to use CatchCalc you will have to know the opposing Pokémon's HP (given that you're not using a master ball). As it stands GameFreak does not give us an option to view the exact HP value of the opposing Pokémon so the best we can do for now is make an estimate, and to play it safe that estimate should be the highest possible value (that is, having 31 IVs in HP). In order to find this value please follow the steps below:

  1. That said, for finding out how much HP the pokemon has that you are encountering you can use any battle simulator (i.e. Pokemon Showdown) or save editor (i.e. PKHeX).

  2. Open up a battle simulator (Pokémon Showdown) or a Pokemon editor (PKHeX / PokeGen etc.)

  3. Pick your Pokémon and set the correct level, then make sure the HP IV is set to 31

    4. Now that you know the maximum possible HP value for the opposing Pokemon you can switch to CatchCalc

  4. Enter the maximum possible HP for Maximum HP

  5. Ideally the "Current HP" will be 1, that is if you used False Swipe or Hold Back. If not then you are going have to make a guess and I cannot help here

  6. Set all your other variables if applicable then press the "Calculate chance to catch" button to have the program do its magic!

FAQ

Q: It doesn't start!
A: Make sure you have associated the .jar extension with Java. To do so right click the program then "open with" and then select the Java platform.

Q: I can't find the Java platform in the list!
A: If you are certain that you did install the above linked JRE, also go ahead and download the Java Development Kit for your OS. Once you have that installed

On Windows:

  • open a command prompt by holding the windows key then pressing R. This will make the Run popup show, in here type cmd.exe
  • in the command prompt type cd [path-to-where-you-saved-the-program] and hit enter
  • then type Java -jar "CatchCalc.jar" including quotation marks!

On Mac (At least this worked for me on my OSX Maverics, I don't use my Mac a lot. at all)

  • Open Terminal
  • Type Java -jar "CatchCalc.jar" including quotation marks!

On Linux (going by Ubuntu here, not sure if it's the same across all distributions)

  • Open Terminal
  • navigate to the folder where the program is, if this is the Downloads folder it will most likely be by typing cd ~/Downloads
  • Type Java -jar "CatchCalc.jar" including quotation marks!
